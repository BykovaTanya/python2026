# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Отрабатываем получение разных отрезков списка, работу с отрицательными индексами и шагом.

## Условие задачи
Ты готовишь шаблон планов для фотосессий. Есть список дат и временных окон. Нужно:
1. Показать первые четыре окна для согласования со стилистом.
2. Получить последние три окна для ночных съемок.
3. Выбрать слоты с индекса 1 по 5, чтобы проверить середину дня.
4. Сформировать срез, который берёт каждый второй слот, начиная с нулевого индекса.

## Пример входных данных
```python
time_slots = ["08:00", "09:30", "11:00", "12:30", "14:00", "15:30", "17:00", "18:30", "20:00"]
```

## Пример выходных данных
```
Утро: ['08:00', '09:30', '11:00', '12:30']
Ночь: ['17:00', '18:30', '20:00']
Дневная серия: ['09:30', '11:00', '12:30', '14:00', '15:30']
Чередование: ['08:00', '11:00', '14:00', '17:00', '20:00']
```

## Критерии успешного выполнения
- Все выборки реализованы через срезы без циклов
- Используются пустые границы, положительные и отрицательные индексы
- Каждый вывод подписан
- Исходный список не мутирует

## Подсказки
- `[:4]` сразу возьмет начало списка
- Хвост удобно брать через `[-3:]`
- Чтобы включить элемент с индекса 5, ставь `stop=6`
- Шаг `2` позволит брать каждый второй слот

## Частые ошибки и ограничения
- Не используй шаг 0
- Следи, чтобы границы не давали пустые срезы
- Не меняй исходный список в процессе
- Не объединяй выборки через `+`, каждое условие выводится отдельно
