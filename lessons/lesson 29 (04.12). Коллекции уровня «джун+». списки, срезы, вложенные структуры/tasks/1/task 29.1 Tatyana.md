# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Закрепляем создание списков, обращение к элементам по индексу и аккуратный перебор без ошибок.

## Условие задачи
Ты собираешь список камер для фотопрогулки: базовые устройства, запасные батареи и карты памяти. Создай исходный список, вставь новую камеру в позицию №2, добавь аксессуары в конец, выведи длину списка, покажи первый и последний элементы. Заверши выводом нумерованного перечня, чтобы быстро проверить сумку.

## Пример входных данных
```python
cameras = ["основная камера", "экшн-камера", "штатив"]
extras = ["доп. батарея", "карта памяти"]
```

## Пример выходных данных
```
Всего предметов: 5
Первый: основная камера
Последний: карта памяти
1. основная камера
2. экшн-камера
3. штатив
4. доп. батарея
5. карта памяти
```

## Критерии успешного выполнения
- Исходный список создан и изменён методами `insert` и `extend`
- Количество элементов определяется через `len`
- Индексы `0` и `-1` применяются к обновлённому списку
- Итоговый список выведен построчно с нумерацией

## Подсказки
- `insert(1, ...)` поставит элемент на вторую позицию
- После `extend` перечень можно сразу печатать через цикл
- Используй `enumerate(..., start=1)`, чтобы не делать ручной счётчик

## Частые ошибки и ограничения
- Не изменяй список внутри цикла выводa, сначала завершай все вставки
- Следи, чтобы список не был пустым перед обращением к индексу `0`
- Не используйте конкатенацию строк вместо списковых методов
- Держи формат вывода консистентным: одна строка — один пункт
