# Задание 29.3. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Мутация и методы списков. Закрепляем добавление, перестановку и удаление элементов, чтобы уверенно управлять реальным списком дел.

## Условие задачи
Ты ведешь расписание репетиций камерного хора:
1. Создай список базовых репетиций на неделю.
2. Добавь через `extend` две дополнительные сессии, присланные хормейстером.
3. Вставь мастер класс по дыханию в позицию №3.
4. Удали по значению репетицию, которую отменили из за болезни солиста.
5. Перемести первый элемент в середину списка с помощью `pop(0)` и `insert`.
6. Выведи итоговый порядок с указанием дня и фокуса репетиции.

## Пример входных данных
```python
rehearsals = ["Пн. строй", "Вт. динамика", "Ср. ансамбль", "Чт. солисты"]
extra = ["Пт. общее прогон", "Сб. работа с дирижером"]
```

## Пример выходных данных
```
1. Вт. динамика
2. Ср. ансамбль
3. мастер класс. дыхание
4. Чт. солисты
5. Пт. общее прогон
6. Сб. работа с дирижером
```

## Критерии успешного выполнения
- Использованы минимум три разных метода изменения списка
- Вставка и удаление выполняются по правильным индексам и значениям
- Перемещение реализовано через `pop` и `insert`
- Итоговый список напечатан после всех операций

## Подсказки
- После `extend` исходный список сразу увеличится
- Индекс 2 вставит элемент как третью позицию
- Чтобы найти середину, используй `len(list)//2`

## Частые ошибки и ограничения
- Не вызывай `remove`, если не проверил наличие элемента
- Не меняй список прямо во время цикла перебора
- Следи за индексами, чтобы не выйти за границы
- Работай только с исходным списком без копий
