# Задание 29.4. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Вложенные структуры. Учимся хранить связанные данные в виде списков словарей и уверенно извлекать вложенные элементы.

## Условие задачи
Ты координируешь фестиваль камерной музыки:
1. Создай список словарей `sessions` с ключами `hall`, `date`, `program` (список словарей с `piece` и `soloist`).
2. Выведи расписание по залам: сначала строку про зал и дату, затем каждое произведение отдельной строкой.
3. Собери список всех солистов без повторов, проходя по вложенным структурам.
4. Построй словарь `soloist -> список произведений`, чтобы быстро находить их репертуар.
5. Отфильтруй залы, в названии которых встречается слово из переменной `focus_word`, и выведи их программы.

## Пример входных данных
```python
sessions = [
    {
        "hall": "Белый",
        "date": "10.12",
        "program": [
            {"piece": "Барокко", "soloist": "Ирина"},
            {"piece": "Модерн", "soloist": "Юлия"}
        ]
    },
    {
        "hall": "Зеркальный",
        "date": "11.12",
        "program": [
            {"piece": "Романтика", "soloist": "Ирина"}
        ]
    }
]
focus_word = "Бел"
```

## Пример выходных данных
```
Зал Белый. дата 10.12
  - Барокко. солист Ирина
  - Модерн. солист Юлия
Зал Зеркальный. дата 11.12
  - Романтика. солист Ирина
Солисты. {'Ирина', 'Юлия'}
Репертуар. {'Ирина': ['Барокко', 'Романтика'], 'Юлия': ['Модерн']}
Залы с фокусом Бел. [{'hall': 'Белый', 'date': '10.12', 'program': [{'piece': 'Барокко', 'soloist': 'Ирина'}, {'piece': 'Модерн', 'soloist': 'Юлия'}]}]
```

## Критерии успешного выполнения
- Данные расписания представлены списком словарей с вложенными программами
- Вывод строится через циклы без хардкода индексов
- Множество солистов формируется автоматически и не содержит дубликатов
- Фильтр по слову не изменяет исходный список

## Подсказки
- Чтобы искать по подстроке, используй `if focus_word.lower() in hall.lower()`
- Словарь репертуара можно собирать через `setdefault`
- Для уникальных солистов достаточно множества

## Частые ошибки и ограничения
- Не добавляй одинаковые имена несколько раз вручную
- Следи за ключами во вложенных словарях, чтобы не получить `KeyError`
- Не перезаписывай `sessions`, когда строишь выборки
- Не выводи пустые структуры. проверяй перед печатью
