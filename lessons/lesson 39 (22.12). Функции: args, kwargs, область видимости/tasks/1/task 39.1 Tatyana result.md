### 6. Вариант полного решения
```python
def mean_abs_deviation(*values):
    if not values:
        raise ValueError("Введите хотя бы один балл")
    avg = sum(values) / len(values)
    deviations = [round(abs(val - avg), 1) for val in values]
    return tuple(deviations)

# Пример использования
if __name__ == "__main__":
    existing_scores = [10, 8, 6]
    print(mean_abs_deviation(*existing_scores))  # (2.0, 0.0, 2.0)
```
