# Задание 33.1. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Введение в генераторы коллекций» — тренируемся заменять повторяющиеся циклы на одно понятное выражение.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Есть список баллов студентов за мини-викторины. Нужно получить новый список строк `"Студент N: passed"`, где N — порядковый номер (от 1), а слово `passed` появляется только если балл ≥ 8. Используй одно выражение list comprehension.

## Пример входных данных
```
scores = [5, 9, 10, 7, 8, 6]
```

## Пример выходных данных
```
['Студент 2: passed', 'Студент 3: passed', 'Студент 5: passed']
```

## Критерии успешного выполнения
- Используется один comprehension с `enumerate`.
- В список попадают только записи с баллом ≥ 8.
- Формат строки сохраняется точно, без лишних символов.
- Исходный список `scores` остаётся неизменным.

## Подсказки
- Вспомни, что `enumerate(scores, start=1)` даёт пары `(номер, значение)`.
- Условие фильтра можно добавить после цикла внутри comprehension.

## Частые ошибки и ограничения
- Не вставляй `append` или дополнительный цикл.
- Не добавляй студентов с баллом < 8.
- Избегай сложных вложенных условий — задача должна читаться с первого взгляда.
