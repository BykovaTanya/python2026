1. **Формулировка задания**  
Требуется создать список `open_groups` с помощью list comprehension. Каждый элемент списка должен быть строкой вида `"<title>: <students> мест"` только для курсов со статусом `"open"` и количеством студентов от 5 до 15 включительно. Исходный список `courses` не должен изменяться.

Файл: `BG33-2.py`  
Код студента:
```python
open_groups = {
    course["title"]: course["students"]
    for course in courses
    if course["status"] == "open"
        if 5< course["students"] <=15
}
print(open_groups)
```

  - Условие `5 < course["students"]` некорректно: должно быть `5 <=` (включительно).
  - Вывод `print(open_groups)` демонстрирует словарь, а не список строк.
