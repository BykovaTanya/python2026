# Задание 33.2. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«List comprehensions» показывает. как за один проход получить новый список и при этом сразу отфильтровать ненужные элементы.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Есть список `courses`, где каждый элемент — словарь с ключами `"title"`, `"students"` и `"status"`. Создай новый список `open_groups`, содержащий строки вида `"<title>: <students> мест"` только для курсов со статусом `"open"` и количеством мест от 5 до 15 включительно. Используй одно выражение list comprehension.

## Пример входных данных
```
courses = [
    {"title": "Python старт", "students": 10, "status": "open"},
    {"title": "Git базовый", "students": 4, "status": "open"},
    {"title": "SQL практика", "students": 12, "status": "closed"},
    {"title": "Async Python", "students": 8, "status": "open"}
]
```

## Пример выходных данных
```
['Python старт: 10 мест', 'Async Python: 8 мест']
```

## Критерии успешного выполнения
- Список `open_groups` формируется одним list comprehension.
- В итог попадают только курсы со статусом `open`, у которых 5 ≤ мест ≤ 15.
- Формат строки точно соответствует образцу.
- Исходный список `courses` остаётся без изменений.

## Подсказки
- Обращайся к полям словаря через `course["title"]` и `course["students"]`.
- Фильтр ставь в конце comprehension с двумя условиями.
- Строки удобно собирать через f-строки.

## Частые ошибки и ограничения
- Не используй дополнительные циклы или `append`.
- Не забудь проверить обе границы количества мест.
- Следи за строгим форматом вывода и пробелами.
