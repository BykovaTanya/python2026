# Задание 33.4. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Set comprehensions» помогает одним выражением получить множество уникальных значений после простой фильтрации.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Есть список `visits`, где каждый элемент — словарь `{ "user": str, "country": str, "device": str }`. Нужно построить множество `mobile_countries`, в которое попадут страны (в нижнем регистре) только тех посещений, где `device` равен `"mobile"` и страна не пустая. Используй одно выражение set comprehension.

## Пример входных данных
```
visits = [
    {"user": "Ann", "country": "Russia", "device": "mobile"},
    {"user": "Bob", "country": "USA", "device": "desktop"},
    {"user": "Cara", "country": "Spain", "device": "mobile"},
    {"user": "Dan", "country": "", "device": "mobile"}
]
```

## Пример выходных данных
```
{'russia', 'spain'}
```

## Критерии успешного выполнения
- Множество создаётся единственным set comprehension.
- Пустые страны не попадают в итог.
- Названия стран приводятся к нижнему регистру.
- Исходный список `visits` остаётся неизменным.

## Подсказки
- Применяй `strip()` перед проверкой, чтобы убрать случайные пробелы.
- Фильтр можно записать как `if visit["device"] == "mobile" and cleaned_country`.

## Частые ошибки и ограничения
- Не оборачивай comprehension в `set()` и не используй `add`.
- Не забудь проверить, что строка страны не пустая после очистки.
- В списках не должно быть больше пяти пунктов.
