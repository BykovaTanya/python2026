## Краткая сводка
- На занятии основные ошибки связаны с отсутствием list comprehension, неправильным форматированием строк и потерей исходной нумерации элементов.
- Цель ДЗ — закрепить требования «один comprehension на задачу», научиться правильно форматировать строки и аккуратно фильтровать данные по условиям.

## Задача 1. Повторение — «Студенты, прошедшие тест»
- **Описание:** дан список баллов `scores = [5, 9, 10, 7, 8, 6]`. Построй список строк `"Студент N: passed"` только для студентов с баллом ≥ 8. Нумерация начинается с 1 и должна сохранять исходные позиции.
- **Критерии:** один list comprehension, используется `enumerate(scores, start=1)`, формат строки строго `"Студент N: passed"`, исходный список не изменяется.

## Задача 2. Практика — «Мини-встречи»
- **Описание:** список кортежей `meetings = [("Standup", 4, 15), ("Demo", 6, 25), ...]`, где кортеж — `(topic, members, minutes)`. Собери список строк `"<topic>: <members> чел, <minutes> мин"` только если `members < 5` и `minutes <= 30`.
- **Критерии:** один list comprehension без `append`, оба условия в конце выражения, результат сохраняется в переменную `compact_meetings`.

## Задача 3. Интеграция — «Карточки отделов»
- **Описание:** есть список словарей:
  ```python
  departments = [
      {"name": "Support", "tickets": 80, "sold": 75},
      {"name": "Analytics", "tickets": 40, "sold": 20},
      ...
  ]
  ```
  - Собери словарь `availability`, где ключ — название города, значение — `"распродан"` если `sold >= tickets`, иначе `"остались места"`.
  - Включай только записи, где `tickets >= 50`.
- **Критерии:** один dict comprehension, тернарный оператор внутри значения, фильтр (`tickets >= 50`) расположен после comprehension, синтаксис без ошибок.

## Задача 4. Подготовка — «Журнал авторизаций»
- **Описание:** список строк `logs = ["anna:login", "bob:update", "claire:logout"]`. Построй множество пользователей, у которых действие `"login"` или `"update"`. Пропускай строки с неправильным форматом (строго одна `:`).
- **Критерии:** один set comprehension, корректное разделение строки, исходный список не модифицируется.
