# Вопрос 17. Тема 3: Рекурсия
  
Что произойдет, если в рекурсивной функции нет условия остановки?
  
Варианты ответов:
1) Python сам остановит
2) ✅ Будет ошибка RecursionError
3) Создаст файл
4) Станет быстрее
5) Выполнится один раз
  
### ИИ анализ
Ответ верный. Без условия остановки функция будет вызывать себя бесконечно, каждый вызов занимает место в стеке вызовов. Когда стек превысит лимит глубины рекурсии (обычно около 1000 вызовов), интерпретатор Python вызовет исключение `RecursionError`.

4) Работа с SQL

2) Только одно ядро
3) Нельзя тестировать
4) Всегда быстрее

### ИИ анализ

Это позволяет эффективнее использовать ресурсы, не блокируя весь поток выполнения в ожидании.  Она не гарантирует ускорения для всех типов задач, особенно для чисто вычислительных.

# Вопрос 39. Тема 6: Основы асинхронного программирования

Варианты ответов:

Ключевое слово `await` приостанавливает выполнение текущей корутины до тех пор, пока переданный ему асинхронный объект (например, другая корутина или задача) не завершится.  Во время этой паузы управление передается обратно в цикл событий, который может запустить другие задачи.  `await` можно использовать только внутри функции, объявленной как `async def`.

3) Всегда процессы
