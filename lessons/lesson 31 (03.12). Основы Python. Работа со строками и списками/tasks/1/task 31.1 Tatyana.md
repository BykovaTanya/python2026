### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая нормализует строку с расписанием одного дня.

Требования:
- прочитать строку формата `урок . предмет` через запятую
- привести каждый предмет к виду с заглавной первой буквой
- удалить лишние пробелы
- вывести каждую пару на новой строке

### Пример входных данных
```
  1 . математика, 2 . пИтон, 3 . Физика 
```

### Пример выходных данных
```
1 . Математика
2 . Питон
3 . Физика
```

### Критерии успешного выполнения
- строка корректно разбирается по запятой
- пары урок. предмет обработаны без потерь
- предметы приведены к нужному регистру

### Подсказки
- используй `split(',')`, потом `split('.')`
- применяй `strip` для удаления пробелов

### Частые ошибки и ограничения
- обработай возможные лишние пробелы вокруг точки
- не используй внешние библиотеки
