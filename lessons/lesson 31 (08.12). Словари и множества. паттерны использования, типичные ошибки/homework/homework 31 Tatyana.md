## Задача 1. Телефонный справочник

Пример входных данных:
```python
entries = [
    ("мама", "+7-900-111-22-33"),
    ("курьер", "+7-999-000-00-00"),
]
request = ["мама", "друг"]
```

### Требуется
1. Создать пустой словарь `phones`.
2. В цикле по `entries` заполнить его, где ключ. имя, значение. телефон.
3. Для каждого имени из `request` напечатать строку `<имя> -> <телефон или "не найдено">`, обязательно используя `get`.

## Задача 2. Суммирование заказов с безопасным удалением

Пример входных данных:
```python
orders = [
    ("Tanya", 3),
    ("Oleg", 1),
    ("Tanya", 2),
]
finished = ["Oleg", "Ivan"]
```

### Требуется
1. Собрать словарь `tickets`, где ключ. имя клиента, значение. общее количество билетов.
2. Для каждого имени из `finished` удалить запись через `pop(name, 0)`.
3. Вывести итоговый словарь и убедиться, что программа не падает, если имени нет в словаре.

## Задача 4. Подготовка к comprehensions

Пример входных данных:
```python
scores = [
    ("task1", 5),
    ("task2", 3),
    ("task1", 4),
]
limit = 5
```

### Требуется
1. Создать словарь `sum_scores` с суммой баллов по каждой задаче.
2. На основе него создать новый словарь только для задач, где сумма не меньше `limit`.
3. Вывести оба словаря.
