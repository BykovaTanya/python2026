### 6. Исправленное решение
```python
rooms = [
    ('piano-1', 'Ирина'),
    ('strings-2', 'Олег'),
    ('drums-1', 'Света'),
]
today_room = 'strings-2'
free_room = 'theory-3'

last_teacher = dict(rooms)

print(f"{today_room} -> {last_teacher.get(today_room, 'свободен')}")
print(f"{free_room} -> {last_teacher.get(free_room, 'свободен')}")
```

**Вывод:**  
```text
strings-2 -> Олег
theory-3 -> свободен
```
