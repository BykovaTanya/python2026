# Вопрос 1. Тема: Форматы JSON и CSV
  
**1. Что из перечисленного НЕ является допустимым типом данных в JSON?**
  
Варианты ответов:
1) Строка
2) Словарь
3) ✅ Функция
  
### ИИ анализ
Правильный ответ — «Функция». JSON — это текстовый формат для обмена данными, и он поддерживает ограниченный набор типов: строки, числа, логические значения, null, массивы и объекты. Функции, классы или другие исполняемые элементы не являются допустимыми типами данных в JSON. Важно помнить, что JSON предназначен для данных, а не для кода.

# Вопрос 2. Тема: Форматы JSON и CSV
  
**2. Какой параметр позволяет улучшить читаемость JSON-файлов при записи?**
  
Варианты ответов:
1) ✅ indent
2) formatter
3) newline
  
### ИИ анализ
Правильный ответ — «indent». В методе `json.dump()` параметр `indent` определяет количество пробелов для отступов, что форматирует вывод JSON с вложенными структурами, делая его удобочитаемым для человека. Параметры `formatter` и `newline` не являются стандартными параметрами модуля `json` для этой цели.

# Вопрос 3. Тема: Форматы JSON и CSV
  
**3. Какой метод используется для записи данных в JSON-файл?**
  
Варианты ответов:
1) json.write()
2) ✅ json.dump()
3) json.save()
  
### ИИ анализ
Правильный ответ — «json.dump()». Метод `json.dump()` предназначен для сериализации объекта Python в JSON-форматированную строку и записи её в файловый объект. Методы `json.write()` и `json.save()` не существуют в стандартном модуле `json` Python.

# Вопрос 4. Тема: Форматы JSON и CSV

1) csv.read()
2) csv.writerows()
3) ❌ csv.dump()
  
### ИИ анализ
Выбран неверный вариант. Правильный метод для записи строк в CSV-файл — это `writerows()` объекта writer, созданного с помощью `csv.writer()`. Метод `csv.dump()` не существует в модуле `csv`, а `csv.read()` используется для чтения данных, а не записи.

# Вопрос 5. Тема: Форматы JSON и CSV

1) ❌ Оба
2) JSON
3) CSV
  
### ИИ анализ

CSV — это плоский табличный формат, который не поддерживает вложенность напрямую; для представления иерархии в CSV требуются дополнительные соглашения.
