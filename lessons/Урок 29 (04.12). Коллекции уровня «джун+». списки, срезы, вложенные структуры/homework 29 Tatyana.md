### Блок 1: Работа со списками (основы)

Ты ведешь список продуктов для пикника. Создай список `products` с тремя продуктами. Добавь два новых продукта в конец списка. Выведи общее количество продуктов, первый и последний продукт. Затем выведи нумерованный список всех продуктов.

Пример входных данных:
```python
products = ["хлеб", "сыр", "колбаса"]
new_products = ["фрукты", "вода"]
```

### Блок 2: Срезы (основы)

У тебя есть список времени встреч с друзьями. 
1. Выведи первые три времени с подписью "Ранние встречи".
2. Выведи последние три времени с подписью "Поздние встречи".
3. Выведи срез со второго по четвертый элемент включительно с подписью "Средние встречи".
4. Выведи каждое второе время, начиная с первого, с подписью "Четные встречи".

Пример входных данных:
```python
meetings = ["10:00", "12:00", "14:00", "16:00", "18:00", "20:00"]
```

### Блок 3: Мутация списков (основы)

У тебя есть список дел на выходные. 
1. Добавь в конец списка дела: "купить подарок" и "позвонить бабушке".
2. Вставь в середину списка (после второго элемента) дело "заправить машину".
3. Удали из списка дело "стирка" (если оно есть).
4. Перемести последнее дело в начало списка.

Пример входных данных:
```python
tasks = ["уборка", "стирка", "готовка", "магазин"]
```

### Блок 4: Вложенные структуры (основы)

У тебя есть список фильмов, каждый фильм представлен словарем с полями: `название`, `год`. 
1. Распечатай информацию в формате: `Название (год)`.
2. Сформируй список названий фильмов, выпущенных после 2010 года.
3. Создай новый список только с фильмами определенного жанра (выбери жанр из списка).
4. Подсчитай количество фильмов.

Пример входных данных:
```python
movies = [
    {"название": "Интерстеллар", "год": 2014},
    {"название": "Матрица", "год": 1999},
    {"название": "Дюна", "год": 2021},
    {"название": "Начало", "год": 2010}
]
```
